# SPE AI Labor Law Assistant

An intelligent chatbot powered by Retrieval-Augmented Generation (RAG) that provides guidance and answers questions about Colombian labor law.

## 📋 Overview

This project implements a conversational AI assistant specialized in Colombian labor legislation. Using RAG technology, the chatbot retrieves relevant legal information from a curated knowledge base and generates accurate, contextual responses to user queries about labor rights, employment regulations, and legal procedures in Colombia.

The system consists of:
- **RAG Pipeline**: Retrieves relevant context from the knowledge base before generating responses
- **Vector Database**: Stores embeddings of Colombian labor law documents
- **LLM Integration**: Generates natural language responses based on retrieved context
- **Chat Interface**: User-facing application for interactions
